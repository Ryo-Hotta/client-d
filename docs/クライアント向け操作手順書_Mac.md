# ホームページ編集マニュアル（Mac版）

> このマニュアルに沿って操作することで、ホームページの内容を自分で編集・公開できるようになります。
> 分からないことがあれば、担当者までご連絡ください。

---

## 全体の流れ

```
① 必要なソフトをインストールする
② ホームページのデータをパソコンに取り込む
③ Cursorで内容を編集する
④ 変更をインターネット上に反映させる
```

---

## 事前準備：GitHubアカウントの作成

GitHubとは、ホームページのデータを管理するサービスです。

1. ブラウザで `https://github.com` を開く
2. 右上の「Sign up」をクリック
3. メールアドレス・パスワード・ユーザー名を入力して登録
4. 登録後、担当者にGitHubのユーザー名を伝える
   ※担当者がホームページへのアクセス権限を付与します

---

## STEP 1：GitHub Desktop のインストール

GitHub Desktopは、ホームページのデータをパソコンに取り込んだり、
編集した内容をインターネット上に送信するためのソフトです。

1. ブラウザで `https://desktop.github.com` を開く
2. 「Download for macOS」をクリック
3. ダウンロードされた `.zip` ファイルをダブルクリックして展開
4. 展開された「GitHub Desktop」アプリを「アプリケーション」フォルダに移動
5. GitHub Desktop を起動
6. 「Sign in to GitHub.com」をクリック
7. ブラウザが開くので、GitHubのメールアドレスとパスワードでログイン
8. 「Authorize desktop」をクリックして連携を許可

> **注意（Mac）：** 初回起動時に「開発元を確認できません」と表示された場合は、
> 「システム設定」→「プライバシーとセキュリティ」→「このまま開く」で許可してください

---

## STEP 2：Cursor のインストール

Cursorは、AIと会話しながらホームページを編集できるソフトです。

1. ブラウザで `https://cursor.com` を開く
2. 「Download for Mac」をクリック
3. ダウンロードされた `.dmg` ファイルをダブルクリック
4. 開いたウィンドウで「Cursor」アイコンを「Applications」フォルダにドラッグ
5. アプリケーションフォルダから Cursor を起動
6. 初回起動時に「Sign In」または「Continue with Google」でログイン
   （Googleアカウントでのログインが簡単です）

> **注意（Mac）：** 初回起動時に「開発元を確認できません」と表示された場合は、
> 「システム設定」→「プライバシーとセキュリティ」→「このまま開く」で許可してください

---

## STEP 3：ホームページデータの取り込み（クローン）

「クローン」とは、ホームページのデータをパソコンに取り込む作業です。
**最初に1回だけ行います。**

1. GitHub Desktop を起動
2. 左上のメニュー「File」→「Clone repository」をクリック
3. 「URL」タブをクリック
4. 入力欄に以下のURLを貼り付ける
   ```
   https://github.com/Ryo-Hotta/client-d
   ```
5. 「Local path」（保存場所）を確認する
   ※ デフォルトのままで問題ありません
6. 「Clone」をクリック
7. 少し待つとダウンロードが完了します

---

## STEP 4：Cursor でフォルダを開く

1. Cursor を起動
2. 「Open Folder」をクリック
3. STEP 3 でクローンしたフォルダを選択して「開く」をクリック
   - 通常は `/Users/（ユーザー名）/Documents/GitHub/client-d` にあります
   - GitHub Desktopの「Show in Finder」ボタンを使うと場所が分かります
4. 左側にファイル一覧が表示されたら準備完了です

---

## STEP 5：AIに話しかけて編集する

Cursor の AI（Composer）を使うと、日本語で話しかけるだけで編集できます。

### AI（Composer）の開き方

キーボードで `Command` + `I` を同時に押す

または、画面上部のメニューから「View」→「Composer」をクリック

### 編集の例

右側に入力欄が表示されるので、例えば以下のように入力します：

```
イベントの開催日を5月3日に変更してください
```

```
お問い合わせページの電話番号を 03-XXXX-XXXX に変更してください
```

```
フッターのイベント情報を「次回イベント：5月3日（土）」に変更してください
```

→ AIが自動でファイルを修正します。内容を確認して「Accept」をクリック

---

## STEP 6：変更をインターネットに反映させる（プッシュ）

編集が完了したら、GitHub Desktopを使って変更を送信します。

1. GitHub Desktop を開く
2. 左側に変更されたファイルの一覧が表示されていることを確認
3. 左下の「Summary」欄に変更内容を一言で入力する
   例：`イベント日程を更新`
4. 青いボタン「Commit to main」をクリック
5. 上部に「Push origin」ボタンが表示されたらクリック
6. 数分後にホームページに反映されます

---

## よくあるトラブルと対処法

### 「Push failed」と表示される
- インターネットに接続されているか確認してください
- GitHubに正しくログインできているか確認してください
- 解決しない場合は担当者にご連絡ください

### Cursorで変更した内容が反映されない
- ファイルを保存しましたか？ `Command` + `S` で保存できます
- GitHub Desktopの左側に変更ファイルが表示されているか確認してください

### 間違えて変更してしまった
- GitHub Desktopで「Discard Changes」を選ぶと元に戻せます
- 分からない場合は担当者に連絡してください（データは必ず復元できます）

### 「xcrun: error」または「git not found」と表示される
- MacにはXcode Command Line Toolsが必要な場合があります
- ターミナルを開いて以下を入力してください：
  ```
  xcode-select --install
  ```
- 画面の指示に従ってインストールしてください
